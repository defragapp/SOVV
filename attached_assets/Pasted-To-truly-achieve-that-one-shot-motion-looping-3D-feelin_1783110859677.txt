# Phase 13: Cinematic Landing (Parallax & Depth of Field)

## Core Objective
Upgrade the `HomePage.tsx` hero section from a flat 2D animation into a continuous, reactive 3D environment. The user should feel like they are holding a physical pane of glass. We will achieve this using Framer Motion's `useMotionValue`, `useTransform`, and `useSpring` tied to mouse coordinates.

---

## Execution Steps

### 1. The Reactive Viewport (Mouse Tracking)
- Wrap the main hero content in a Framer Motion `<motion.div>`.
- Implement `useMotionValue` to track the `clientX` and `clientY` of the user's pointer.
- Pass these values through a `useSpring` hook (e.g., `stiffness: 50, damping: 20`) to give the movement physical weight and smooth out the data.

### 2. 3D Transforms & Parallax Layers
- Use `useTransform` to map the smoothed mouse coordinates to `rotateX` and `rotateY` properties on the main container (range: `-2deg` to `2deg`).

### 3. Tactile Glass CTAs
- Upgrade the "Enter Sovereign.os" CTA. 
- Do not use standard CSS hover transitions. Wrap it in a `<motion.button>`.
- **While Hover:** Add a subtle `scale: 1.02` and intensify the hairline border opacity.
- **While Tap:** Compress it sharply (`scale: 0.97`) to mimic the physical haptic snap of an iOS button. 

Execute these Framer Motion physics on `HomePage.tsx`. Ensure it degrades gracefully to a static view on mobile or when `prefers-reduced-motion` is active.
